::: info 前言
在2024年5月的一个寻常夜晚，窗外的风轻轻吹过，笔者本以为这只是初中生活中千篇一律的一天。然而，命运的齿轮在那一刻悄然转动，因为 [@liyao](http://liyao.edu.kg) 的一句话，我翻开了与网站建设的第一页，同时也彻底翻开了我与互联网世界精彩纷呈的第一页……
:::



一切的开始源于 [@liyao](http://liyao.edu.kg) 随口的一句提点，他告诉笔者他拥有属于自己的域名。这句话在笔者心中掀起了惊涛骇浪——原来“域名”这种听起来高不可攀、仿佛只存在于互联网巨头手中的名词，竟然也是一介凡人触手可及的物品！

::: tip 小提醒
这种认知的冲击让我兴奋不已。于是，怀揣着忐忑与激动，我注册了我的第一个域名。当屏幕上显示注册成功的刹那，那种拥有属于自己网络领地的成就感油然而生。
:::

也是在同一个月，我拥有了第一个QQ号，开始正式地、不再是旁观者般地接触互联网……



就在不久后，贪婪的“淘金”本能让我在[硅云](https://www.vpsor.cn)发现了 `.top` 域名首年零元的促销活动——对于当时的我来说，这简直是天上掉馅饼。**没错，笔者毫不犹豫地拿下了它**，也就是现在用于承载各类服务的域名。不过，那时的我还是个懵懂的菜鸟，并不懂得备案为何物，因为手中没有服务器，备案的事宜便搁置了下来，直到第二年转入阿里云时才补上。



依然是那股子初生牛犊不怕虎的劲头，不久后，我又在华为云国内站看见了首年一元的 `.com` 域名活动——**没错啊，笔者又没忍住下手了**。不过，这个域名目前已经到期，当时的促销活动也早已无了。当看到高达 **98元人民币** 的续费价格时，笔者这位还在读书的初中生感到了深深的无力感。现在回过头想想，8+3个字符的域名，既冗长又难记，确实不够完美。况且域名这种东西，贪多嚼不烂，囤积太多其实并没有实际用处 ~~虽然笔者现在已经拥有了3个正价、3个特价和4个赠送的域名了。~~



既然说到了域名，就来盘点一下笔者的“家底”吧！

回到现实中，依然是那个无法回避的问题——由于笔者是初三在读生，并没有独立的经济能力来续费以上全部域名（共计￥293.42）。这是一个关于取舍的痛苦决定：到期后，我只能忍痛割爱，保留 **be\*-\*\*.cn、mfawa.top、imbee.top、byoip.top、1s.fan、368683.xyz、19931110.xyz**。即便如此精简，这笔费用仍然高达 **￥206**，对于一个学生党来说，咬咬牙从压岁钱里挤一挤，倒也是拿得出的，只是钱包要消瘦好一年了。



好像有些偏题了，让我们把时间轴拨回来。在2024年8月30日，笔者在群聊中结识了 @mxdyeah。至于相知的细节嘛……不知道 [@mxdyeah](http://www.mxdyeah.com) 是否允许公开，所以暂时打个马赛克保密:D，但他绝对是笔者技术道路上的一位贵人。如果说 [@liyao](http://liyao.edu.kg) 是那个为我打开大门的引路人，那么 [@mxdyeah](http://www.mxdyeah.com) 则是带我飞跃山丘、看到更广阔世界的导师。



后来，笔者在另一个群里，偶然看到一位群友发出了TA的个人主页链接（kldhsh.top），当然不是现在这版。点进去的那一刻，我滴个乖乖，这简直是按照笔者的审美量身定制的——**那柔和的圆角，那丝滑的过渡动画，那恰到好处的布局，简直了！**那种视觉冲击力让我瞬间沦陷。笔者急忙火急火燎地询问群友：“这源码哪里搞的？（注：当时笔者甚至还不了解vue是什么）”。群友爽快地甩出了Github仓库的URL（[imsyy/home](http://github.com/imsyy/home)）。于是，笔者在千恩万谢了群友后，迫不及待地投身到研究 “[@imsyy/home](http://github.com/imsyy/home)” 的工作中去，越看越啧啧称奇——这也太好看了吧！简直就是艺术品！



然而，由于一开始，笔者连 `vue`、`npm` 这些基础概念都一头雾水，对着黑底白字的终端窗口搞了许久，除了报错什么成果都没有。夜色渐深，困意袭来，无奈只能先放一边去睡觉了~~（还是没忍住刷了半小时视频）~~

::: warning 踩坑警告
第二天中午，笔者顶着黑眼圈去求助群友。

> “大佬，我该咋弄？”
> 
> “你先去下载个Node.js吧。”
> 
> “装好了，然后呢？”
> 
> “然后就按照README文件里的指示，照猫画虎跟着做即可。”
> 
> “好，我去试试，谢谢！”

接着，笔者就按照README开始输入 `pnpm install` —— **“pnpm 未找到该命令？”**。笔者当时心里一紧，也不大敢去问群友（怕被嫌弃是啥都不懂的小白），只能灰溜溜地去搜索引擎查——“哦……原来要先用 `npm install -g pnpm` 全局安装了pnpm才行”。恍然大悟后再次输入 `pnpm install`，新的问题又来了：“咦？下载怎么这么这么慢呐？进度条像蜗牛一样”。这次实在忍不住了，厚着脸皮去问群友：“大佬，我说pnpm下载怎么这么慢啊？”群友秒回：“你去Bing一下‘pnpm换源’。”笔者：“行，我试试。”……经过一番折腾，终于看到终端里跳出了构建成功的提示。笔者那一刻简直想跳起来，连忙去群里报喜：“我c，终于好了，谢谢大佬~”
:::

虽然过程略有曲折，充满了踩坑的辛酸，但结果总归是甜的。笔者就是在这那一刻，深深地爱上了 `vue.js`，并由此，喜欢上了 `Node.js` 和 `nuxt.js` 生态。




尝到甜头后，我又接着发现了[imsyy](http://github.com/imsyy)大大写的其他网站项目。笔者看得两眼放光——每一个网站都精准地长在了笔者的审美点上。于是，贪心大发的笔者打算把它们全部部署一遍。有了构建 [@imsyy/home](http://github.com/imsyy/home) 的经验，其他的构建起来也变得得心应手，不一会儿就全部搞定了，然后推送到Github，托管到Vercel。看着自己的服务一个个上线（imbee.top），那种满足感无以言表。



在此后的日子里，生活略显平淡，笔者开始觉得有些许无聊，漫无目的地翻阅着视频网站。偶然间，我触碰到了“无畏契约”（瓦）。当时还没有“手瓦”（玩游戏），只是作为云观众。随着了解的深入，我开始结识瓦友，学习压枪、练烟——短暂地沉迷在游戏给笔者带来的多巴胺快乐中。要说笔者不务正业，倒也不是绝对的，玩确实是玩了，但笔者也维护着网站的基本运行，奇怪的是，成绩竟然也没往下掉（也许是因为已经差到无可救药了？）。



游戏初期给笔者带来的快乐确实不少——**拿到五杀时的欣喜若狂、队友的默契夸赞、逆风翻盘时在语音里的欢呼**……那是一段热血沸腾的时光。不过，笔者最终还是选择了退游。当一个游戏火出了圈，它就会吸引无数圈外人涌入——这本该是件好事，新鲜的血液注入，活跃了气氛……但是，这也吸引了无数的外挂、XXS（指没有素质、没有理智的人），以及无数的带妹炸鱼、压力怪。



游戏环境肉眼可见地变差了。官方虽然有在管理，但代价是像笔者这种老实本分的“绿玩”受到的影响最大。ACE暴力误封、恶意炸鱼、无端谩骂，一时间乌烟瘴气充斥着“无畏契约”。更让人心寒的是，在现实生活中，由于游戏圈的一些负面舆论，我们这些玩家也被圈外人贴上了“瓦学弟”、“性压抑”的标签。笔者亲身经历过看别人乱认妈、恶意骚扰女玩家等等恶心事件。



污染游戏环境的不仅有坏人，但也有许多想认认真真玩游戏、来瓦休闲娱乐的正常玩家，却无端遭受攻击。笔者可以想象，看着自己所热爱的游戏环境一天不如一天，在现实生活中被人指指点点，那种无力感……无奈之下，只能被迫退游。

::: danger 退游导火索
而让笔者彻底决裂的导火索，其实是**官方的一次平衡性更新**，这成为了压死骆驼的最后一根稻草——**几乎全部笔者喜爱的特工被大幅削弱，甚至连地图的改动也不合我意**。这让笔者积怨已久的不满彻底爆发：

> “这瓦，谁爱玩谁玩，我不奉陪了！”

于是乎，笔者毅然卸载，重拾了网站建设这一更纯粹的爱好。
:::



从退游至今（2026/1/17），笔者已有几个月的时间彻底没碰过游戏了。原本用来打游戏的时间，现在被网站开发和社群填满。不过这是笔者的真爱，当你做的事情是你所热爱的，再累也会觉得觉得值得。（正如那句老话：**男女搭配，干活不累**doge）



时间来到2025年7月，笔者开始不满足于仅仅使用纯HTML写简单的静态页面。在短时间内，我像海绵吸水一样疯狂了解更多网站架构、开发的相关知识，可以说是**恶补了沉迷游戏期间落下的技术债**。在此期间，我也加入了许许多多的技术交流群，结识了许多志同道合的群友。

::: info 社群收获
在这些社群中，让笔者印象最深刻的是**“瑞白云交流群”**和**“二叉树技术交流群”**。通过这两个群认识的大佬最多，学到的“偏方”（实用窍门）也最多，比方说 CF Worker优选IP、Netlify Function的妙用、不同网络线路的差异数据分析等等。

在这些社群中，让笔者印象最深刻的是“瑞白云交流群”和“二叉树技术交流群”。通过这两个群认识的大佬最多，学到的“偏方”（实用窍门）也最多，比方说 CF Worker优选IP、Netlify Function的妙用、不同网络线路的差异数据分析等等。在那里，我也能看到许多灰度小白遇到的问题，恰好是我刚刚踩过的坑，比如 Vercel部署错误、SaaS优选配置、CDN规则复杂的写法等，我也乐于分享我的经验。



在“瑞白云交流群”，我所认识的群友中，令我记忆犹新的是 **@Creeper亦缺**、**@Xiaolan** 和 **@htyuty**。说来也好笑，认识的过程简直是由一大堆“恰好”碰在一起。当时 @亦缺 需要一个活跃度较高的Github账号来达成某种成就或合作，而笔者这几天正好因为沉迷研究网站，Github提交绿点非常频繁。顺理成章地，笔者与 @Creeper亦缺 达成了合作。至于 @Xiaolan 的认识过程，时光有些模糊了，不过依稀记得是因为一起测试某条CDN线路的延迟？算了，具体细节忘记就忘记吧，重要的是相识。



在“二叉树技术交流群”，我认识的朋友算是比较多的了，比如 **@fishcpy**、**@夜轻**、**@wyf9**、**@yunsen2025** 等……这些个朋友的相识过程就更具戏剧性了。@fishcpy 当时只是在群里闲聊，笔者闲来无事，随手去加了他的好友——竟然通过了！后来发现 @fishcpy 搞的CF优选特别好，技术过硬，正好笔者也在搞优选（只不过没公开），两人一拍即合，笔者就与 @fishcpy 合作搞了个“**cnae / NB优选**”网站，就这样成了战友。



至于 @wyf9？笔者当时看到这个名字惊喜了一下：“**哇！跟我名字的首字母一模一样欸！这世界太小了！**”然后一聊，得知他竟然是一群之主，还是SiiWay的创始人之一，而且比笔者也大不了多少？！这种同龄人的共鸣感让我兴奋不已，于是，笔者毫不犹豫地加入了他的团队。现在每天都在群里吹水聊天，分享新发现，嘿嘿。



就目前而言，笔者主要活跃在“二叉树技术交流群”、“SiiWay 官方交流群”和“枫屿/杂鱼团队”。每天在群里吹吹水、聊聊天；交流技术心得，今天他发现了个好用的API，明天笔者发现了个好用的脚本……

哦，对了，差点忘了说，“枫屿/杂鱼团队”是技术大神 **@xiaolan** 创办的，这也是个充满活力的大家庭。

> **愿我们都能在热爱的道路上，继续前行。**
